# 🎮 Pokémon FireRed - Web Emulator

Este projeto é uma interface web simples e funcional para jogar **Pokémon FireRed** diretamente no navegador, utilizando a biblioteca **EmulatorJS**.

---

## 🕹️ Controles

O emulador já vem pré-configurado com os seguintes comandos:

| Tecla | Ação (GBA) |
| :--- | :--- |
| **Z** | Botão A |
| **X** | Botão B |
| **Enter** | Start |
| **Shift** | Select |
| **Setas** | Movimentação |

---

## 🚀 Funcionalidades

- 🎮 **Auto-carregamento**: O jogo inicia automaticamente ao abrir a página  
- 💾 **Save State**: Suporte para salvar e carregar o progresso  
- ⚡ **Core GBA**: Motor otimizado para Game Boy Advance  
- 🧼 **Interface limpa**: Foco total na experiência do jogador  

---

## 🛠️ Tecnologias Utilizadas

- HTML5 / CSS3  
- JavaScript  
- [EmulatorJS](https://www.emulatorjs.com/) (Motor de emulação)

---

## ⚙️ Como rodar o projeto

1. Clone este repositório:
```bash
git clone https://github.com/joaooliveiralino/Pokemon-FireRed-Web-Edition.git
```

2. Acesse a pasta do projeto:
```bash
cd pokemon-web
```

3. Certifique-se de que a ROM está no caminho:
```
roms/pokemon.gba
```

4. Execute o projeto em um servidor local (necessário por causa de CORS):

## ▶️ Opção 1 — VS Code (recomendado)
- Instale a extensão **Live Server**
- Clique com o botão direito no `index.html`
- Selecione **Open with Live Server**

## ▶️ Opção 2 — Python
```bash
python -m http.server
```
